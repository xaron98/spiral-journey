# Spiral mockups v2 (basados en captura real)

Estos mockups se diseñaron tomando como referencia la pantalla real compartida por el usuario:
- Header de bienvenida + fecha.
- Spiral central como protagonista.
- Tarjetas informativas en bloque inferior.
- Tab bar flotante.

## Conceptos
1. `spiral-liquid-glass-home.svg` — Home minimal con superficies glass translúcidas.
2. `spiral-liquid-glass-focus.svg` — Prioriza lectura de espiral y reduce ruido cromático.
3. `spiral-liquid-glass-coach.svg` — Variante orientada a insights/coach con jerarquía fuerte.

## Principios aplicados
- Simplicidad (menos elementos simultáneos).
- Limpieza visual (espaciado amplio + tipografía sobria).
- Innovación (capas difusas, reflejos suaves y contraste local tipo liquid glass).
